# 🚀 دليل M-Store الذكي (Autonomous System)

تم تطوير M-Store ليكون نظاماً ذاتي الإدارة بالكامل. الآن يمكنك إضافة تطبيقاتك دون تعديل أي ملف JSON يدوياً.

## 🛠️ كيف تضيف تطبيقاً جديداً؟

### 1. إنشاء مجلد التطبيق
داخل مجلد `apps/` الرئيسي، أنشئ مجلداً جديداً باسم التطبيق (ID).

### 2. هيكلة الملفات (تلقائي بالكامل)
قم بوضع ملفاتك كالتالي:
```
/my-new-app
├── app.json (بيانات وصفية أساسية)
├── icon.png (أو jpg, webp, svg)
├── banner.png
└── /screenshots (ضع أي عدد من الصور هنا)
└── /assets (ضع أي صور إضافية للمميزات هنا)
```

### 3. إعداد ملف app.json
تحتاج فقط للبيانات النصية، الصور يتم اكتشافها تلقائياً.
```json
{
  "id": "my-new-app",
  "name": "اسم التطبيق",
  "category": "Tools",
  "version": "1.0.0",
  "description": "وصف قصير",
  "longDescription": "وصف مفصل",
  "rating": 5.0,
  "size": "25 MB",
  "android": "8.0+",
  "downloadUrl": "رابط التحميل"
}
```

### 4. تحديث الفهرس (Manifest)
قبل رفع التغييرات، قم بتشغيل سكربت التحديث التلقائي:
```bash
node scripts/generate-manifest.js
```
سيقوم السكربت بمسح كل المجلدات واكتشاف الصور ولقطات الشاشة وتحديث المتجر فوراً.

## 📊 عداد التحميلات (Firebase)
النظام متصل بقاعدة بيانات Firebase Realtime.
- يتم تحديث العداد تلقائياً عند ضغط المستخدم على زر التحميل.
- يتم عرض أكثر التطبيقات تحميلاً في قسم **Trending** تلقائياً.

## 🎨 الهوية البصرية
- يدعم النظام جميع صيغ الصور (WebP, PNG, JPG, SVG).
- يتم تطبيق **Lazy Loading** لضمان سرعة فائقة.
- في حال فقدان أي صورة، يظهر **Placeholder** احترافي تلقائياً.
